## Step 9: FC Online 조작으로 변경 ✅
> 기존: 방향키+S/A/D/W/Space → FC Online 스타일로 전면 개편
- [x] 계획 수립 (검토 완료)

### 9-1. E키 전력질주
- [x] E 누르면 선수 speed 1.5배, 떼면 원래 속도
- [x] E 질주 시 공이 발에서 살짝 앞으로 튐

### 9-2. A키 상황별 분리
- [x] 공격(공 있음): A = 로빙패스/크로스
- [x] 수비(공 없음): A = 슬라이딩 태클 (미끄러지며 공 뺏기)

### 9-3. C키 등지기/견제
- [x] 공격(공 있음): C = 등지고 볼 보호 (천천히, 주변 수비 밀어내기)
- [x] 수비(공 없음): C = 견제 자세 (느리게 이동)

### 9-4. Q키 수비 압박
- [x] Q 누르면 가장 가까운 AI 선수가 공 쪽으로 전력 질주
- [x] Q 떼면 원래 포지션 복귀

### 9-5. Z+D 감아차기 / Q+D 칩슛
- [x] Z 누른 채 D = 감아차기 (휘어짐)
- [x] Q 누른 채 D = 칩슛 (높은 궤적)

### 9-6. Shift+방향키 개인기
- [x] Shift+방향 = 해당 방향으로 급속 전환 (0.4초 쿨)

## Step 10: 파워 게이지 (FC Online 스타일)
- [x] D/S/A/W 키를 꾹 누르면 조작 선수 위에 게이지 바 표시

- [x] 키를 떼면 게이지 %만큼의 파워로 액션 실행
